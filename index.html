      About This Panel

      Real-time network connectivity and offline time monitoring for all in-store devices.

      • P UP / P Down — Primary ISP connection status (P Down = orange)
      • B UP / B Down / B None — Backup ISP connection status (B Down = yellow)
      • Card border color — Green = active · Orange = on backup · Red = down
      • ▲ triangle — A sub-device (switch, cash register, admin PC) has an issue
      • Click a card — Opens full device detail for that store
      • Offline Time Report — Downtime per category plus Primary/Backup WAN % (store hours, refId X/Y in the panel)

      Summary card meanings:

      • Active — Connected normally via primary link
      • Inactive — All connections are down
      • On Backup — Running on backup (primary down)
      • Backup Down — Backup configured but unreachable
      • No Backup — No backup ISP configured
